  <name>Task 1: Establish TSL Material Foundation</name>
  <files>src/Factory.js, src/TextureGenerator.js</files>
  <action>
    Convert the material creation logic in `src/Factory.js` and `src/TextureGenerator.js` to use `NodeMaterial` and TSL nodes.
    
    Implementation details:
    - Import `MeshPhongNodeMaterial` (or `MeshStandardNodeMaterial` where PBR is preferred) from `three/webgpu`.
    - Import TSL nodes like `texture`, `color`, `float`, and `vec3` from `three/tsl`.
    - Update `TextureGenerator.js` to ensure the generated `CanvasTexture` works correctly when assigned to `material.colorNode = texture(tex)`.
    - Systematically replace `new THREE.MeshPhongMaterial({ map: tex })` with:
      ```javascript
      const material = new MeshPhongNodeMaterial();
      material.colorNode = texture(tex);
      ```
    - Handle color and shininess properties using TSL nodes (e.g., `material.shininessNode = float(80)`).
  </action>
  <verify>
    <automated>npm run build</automated>
  </verify>
  <done>Core materials converted to TSL NodeMaterials.</done>
</task>
