  <action>
    Add `getGaussian(stdDev = 1)` to the `Utils` object in `src/Utils.js`.
    Use the Box-Muller transform:
    ```javascript
    getGaussian(stdDev = 1) {
        let u = 0, v = 0;
        while(u === 0) u = Math.random(); 
        while(v === 0) v = Math.random();
        return Math.sqrt(-2.0 * Math.log(u)) * Math.cos(2.0 * Math.PI * v) * stdDev;
    }
    ```
    This will be used for bullet distribution within the weapon spread radius.
  </action>
