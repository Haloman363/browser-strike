    Refactor `WeaponSystem` to handle deterministic recoil and movement inaccuracy.
    - Track `this.shotIndex` (init to 0).
    - In `setFiring(isFiring)`: If `!isFiring`, reset `this.shotIndex = 0`.
    - In `shoot()`:
        1. Query `PlayerControllerSystem` for `velocity` and `isCrouching`.
        2. Calculate `totalSpread`: `baseSpread + (velocity.length() * moveInaccuracy)`. If crouching, apply a 0.5x multiplier.
        3. Increment `this.shotIndex`.
        4. Apply `recoilPattern[this.shotIndex % pattern.length]` to `cameraRecoilX` (vertical) and `cameraRecoilY` (horizontal/yaw).
        5. Apply `totalSpread` using `Utils.getGaussian()` to the shot direction vector before raycasting/sending to network.
    - In `update()`:
        - Implement exponential recovery for `cameraRecoilX`, `cameraRecoilY`, and `this.currentSpread`.
        - Expose `this.currentSpread` as a public property for `UISystem`.
  </action>
