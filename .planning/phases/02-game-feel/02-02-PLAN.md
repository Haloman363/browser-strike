<task type="auto" tdd="true">
  <name>Task 2: Client-Side Reconciliation Loop</name>
  <files>src/systems/NetworkSystem.js, src/systems/PlayerControllerSystem.js</files>
  <behavior>
    - In `NetworkSystem.update`:
      - Skip interpolation for the local player entity (ID matching).
      - If a snapshot contains state for the local player:
        - Extract `lastSeq` from the snapshot.
        - Call `playerController.reconcile(authoritativeState)` with the snapshot data.
    - In `PlayerControllerSystem.reconcile(state)`:
      - Filter `inputBuffer`: remove all inputs with `seq <= state.lastSeq`.
      - Check distance between local position and `state.pos`.
      - If distance > 0.1:
        - SNAP local position to `state.pos`.
        - REPLAY all remaining inputs in `inputBuffer` using `applyInput`.
  </behavior>
  <action>
    1. Modify `NetworkSystem.update(dt)`:
       - Identify the local player entity (use `this.peer.id` or similar).
       - Skip `SnapshotInterpolation` logic for the local player.
       - If the latest snapshot has state for the local player, pass it to `PlayerControllerSystem.reconcile`.
    2. Implement `PlayerControllerSystem.reconcile(authoritativeState)`:
       - Update `inputBuffer = inputBuffer.filter(i => i.seq > authoritativeState.lastSeq)`.
       - Perform the desync check (threshold 0.1).
       - If desync found, set position to authoritative and loop through `inputBuffer`, calling `applyInput`.
  </action>
  <verify>
