<task type="auto" tdd="true">
  <name>Task 2: Implement Input Buffering and Networked Transmission</name>
  <files>src/systems/PlayerControllerSystem.js</files>
  <behavior>
    - Each frame, capture `inputState` (keys: W,S,A,D,Space,Ctrl).
    - Increment `sequenceNumber`.
    - Buffer input with its sequence number and delta: `{ seq, delta, keys }`.
    - Send input packet to the host via `NetworkSystem.send('INPUT', data, false)`.
    - Local prediction must use the buffered input immediately.
  </behavior>
  <action>
    1. Add `this.inputBuffer = []` and `this.sequenceNumber = 0` to constructor.
    2. In `update(delta)`:
       - Increment `sequenceNumber`.
       - Capture current keys from `InputSystem`.
       - Create `inputState = { seq: this.sequenceNumber, dt: delta, keys: { ... } }`.
       - Push to `this.inputBuffer`.
       - Call `applyInput(inputState, delta)`.
       - Call `this.engine.getSystem('NetworkSystem').send('INPUT', inputState, false)` (unreliable channel).
    3. Ensure input is NOT predicted if player is dead or planting (existing constraints).
  </action>
  <verify>
    <automated>npm test src/systems/PlayerControllerSystem.test.js</automated>
  </verify>
  <done>Inputs are buffered, sequence-numbered, and sent to host while predicting locally.</done>
