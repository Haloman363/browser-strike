# Phase 14: Gameplay Mechanics - Plan 01: Advanced Economy & Buy System

**Objective:** Implement a robust tactical economy system with consecutive loss bonuses, specific kill rewards, and utility buy limits.

## Key Goals
1. **Dynamic Loss Bonus**: Implement a "Loss Streak" counter that increases the round loss reward ($1400 -> $1900 -> $2400 -> $2900 -> $3400).
2. **Specific Kill Rewards**: Categorize weapons and grant specific rewards (e.g., Knife $1500, SMG $600, AWP $100).
3. **Utility Limits**: Enforce limits on the number of grenades a player can carry (max 4 total, max 2 flashbangs).
4. **Economic HUD**: Enhance the UI to clearly show current loss bonus and estimated next-round cash.

## Proposed Changes

### `src/GameState.js`
- Add `lossStreak: 0`.
- Add `utilityCount: { he: 0, flash: 0, smoke: 0, molotov: 0 }`.

### `src/systems/RoundSystem.js`

    - Reset `lossStreak` for the winning team.

### `src/systems/WeaponSystem.js`

### `src/Constants_v2.js`

### `src/UI.js` or `src/systems/UISystem.js`

## Verification Plan
1. **Economy Test**: Lose multiple rounds in a row and verify the money granted increases correctly.
2. **Reward Test**: Kill an enemy with a Knife and verify $1500 is granted.
3. **Limit Test**: Attempt to buy 3 flashbangs and verify the system blocks the 3rd purchase.
